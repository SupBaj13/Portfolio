# FOCP – Weekly Exercises

This repository contains weekly exercise files for the *FOCP* course. Each folder represents a specific week and includes a `.docx` file with course exercises and questions.

## Project Structure
- Week 1 – Exercises 01.docx  
- Week 2 – Exercises 02.docx  
- Week 4 – Exercise 04.docx  
- Week 5 – Exercises 05.docx  
- Week 6 – Exercise 06.docx  
- Week 7 – Exercise 07.docx  
- Week 8 – Exercises 08.docx  

## Topics Covered
- Programming fundamentals  
- Variables and data types  
- Conditional statements  
- Loops  
- Functions  
- Problem-solving techniques  

## How to Use
Open the `.docx` files using Microsoft Word, LibreOffice Writer, or Google Docs.
