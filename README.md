This repository contains coursework materials for the *Fundamentals of Computer Programming (FOCP)* module. The content is organized into folders based on activity type, including weekly logbooks, Python practice files, and final project programs.

## Repository Structure

- **logbook**
  - Contains weekly logbook files in `.docx` format
  - Includes exercise questions and written practical tasks for each week

- **notebook**
  - Contains weekly Python (`.py`) files
  - Covers practical exercises and programs from Week 1 to Week 8

- **project_programs**
  - Includes final assignment solutions:
    - Task 1
    - Task 2
  - Demonstrates application of core programming concepts in complete programs

## Learning Focus
- Python programming fundamentals
- Variables, data types, and Boolean logic
- Conditional statements and loops
- Functions and basic computations
- Problem-solving and structured coding practices

## How to Use
- Open `.docx` files from the **logbook** folder using Microsoft Word, LibreOffice, or Google Docs
- Run `.py` files from the **notebook** and **project_programs** folders using any Python-supported IDE or editor
